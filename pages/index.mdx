# Zotistics

<Callout type="warning">
  This site contains developer documentation for Zotistics. If you're looking
  for help as a user, please visit the
  [FAQ](https://beta.zotistics.com/posts/faq). If you are looking to link to

Zotistics is a simple, accessible interface for viewing historical grade data. It allows students and professors to compare their classes against department averages and other instructors, helping them make more educated decisions.

## Setup

1. Clone the Zotistics repository.

```bash
$ git clone https://github.com/icssc-projects/Zotistics.git
```

2. Navigate to the root directory and install its dependencies.

```bash
$ cd Zotistics
$ npm install
```

3. Start the development server.

```bash
$ npm run dev
```

4. Zotistics should load locally at http://localhost:3000/
